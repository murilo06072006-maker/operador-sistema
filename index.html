Princípios

Consistência

  • Presença diária supera esforço esporádico
  • 20 min/dia de estudo > 4h no fim de semana
  • Treino leve presente > treino perfeito ausente
  • Registrar cria comprometimento real

Progressão

  • Carga, volume, habilidade: aumentar gradualmente
  • Medir para gerenciar
  • Deload não é retrocesso, é parte do ciclo
  • Portfólio cresce linha por linha

Mentalidade SOC

  • Defender é ativo, não reativo
  • Todo log conta uma história
  • Documentar é operar profissionalmente
  • Adversário pensa em cadeia; você também

Identidade

  • Analista, atleta, estudante: as três frentes andam juntas
  • Corpo forte suporta mente disciplinada
  • Graduação é base, não destino
  • Cada dia de execução define quem você é
